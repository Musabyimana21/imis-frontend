# IMIS cPanel Deployment Instructions 
 
## Backend Deployment: 
1. Upload backend folder to public_html/api/ 
2. Install Python packages: pip install -r requirements.txt 
3. Set up Python app in cPanel pointing to app/main.py 
 
## Frontend Deployment: 
1. Upload frontend contents to public_html/ 
2. Update API URLs to point to https://e-shakiro.com/api
