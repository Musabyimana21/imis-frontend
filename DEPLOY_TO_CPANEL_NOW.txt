STEP 1: BACKEND DEPLOYMENT (5 minutes)
=====================================

1. In cPanel, go to "Python App"
2. Click "Create Application"
3. Settings:
   - Python Version: 3.9 or higher
   - Application Root: api
   - Application URL: /api
   - Application Startup File: app/main.py
   - Application Entry Point: app

4. Upload backend files:
   - Upload ALL files from cpanel_deploy/backend/ to /home/eshakiro/api/
   - Make sure .env file is uploaded

5. Install dependencies:
   - In Python App, click "Open Terminal"
   - Run: pip install -r requirements.txt

6. Start the application


STEP 2: DATABASE SETUP (3 minutes)
==================================

1. In cPanel, go to "MySQL Databases"
2. Create database: eshakiro_imis_production
3. Create user: eshakiro_imis_user
4. Password: Bachjudoly@11996
5. Add user to database with ALL PRIVILEGES

6. Run database setup:
   - In Python App terminal, run: python setup_db.py


STEP 3: FRONTEND DEPLOYMENT (2 minutes)
======================================

1. Upload ALL files from cpanel_deploy/frontend/ to /home/eshakiro/public_html/
2. Make sure .htaccess file is uploaded for proper routing
